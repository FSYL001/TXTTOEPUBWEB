# 🌸 TXTtoEPUBtoolWEB

> 风神翼龙的TXT转EPUB网页 — **可逆字符加密 & 解密**  
> 一个纯前端的字符加密小工具 · 离线可用  
> 🛠️ 主要是自己写着玩的，练手小作品，防止以后找不到就丢 GitHub 备份 😂

---

## ✨ 功能
- ⚡ 纯前端加密 / 解密，无需服务器  
- 🔤 支持 TUF-8 的TXT  
- 🧩 可预览文本自定义文本内容 书名 作者 封面 识别章节
- 🖥️ 响应式设计，PC / 手机 / 平板均适配  
- 🎯 简单易用，零基础可操作
